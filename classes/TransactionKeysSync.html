    processMempoolTransaction

    • processMempoolTransaction(txHash: string, rawTx: string): Promise<void>

      Process a mempool (unconfirmed) transaction using the same ownership detection logic as confirmed blocks. Outputs owned by this wallet are stored with blockHeight=0 and inputs that spend wallet outputs are marked as spent with spentBlockHeight=0.

      The raw transaction hex is deserialized locally to extract output keys and range proofs, avoiding round-trips to ElectrumX which may not serve mempool output data.

      Parameters

      • txHash: string

        The transaction hash (as returned by broadcast)

      • rawTx: string

        The serialized transaction hex

      Returns Promise<void>
